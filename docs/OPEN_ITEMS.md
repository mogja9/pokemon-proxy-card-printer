- A4 + bleed + gutter (RESOLVED 2026-06-11, NO code change - by design): on A4 with the default 4mm gutter the synthesized 3.175mm bleed does reach ~3.325mm from the page edge, inside the 6.35mm unprintable margin. What must stay printable is the trim block + crop marks, and contentFitsPaper already guards that (e.g. A4 + 8mm gutter -> x0=2.5mm < 6.35 -> it warns). bleedFitsPaper('A4')==true is therefore correct; no fix needed.
